# Model Deploy Platform

Independent self-service model deployment control-plane.

## Quick Start

```bash
make test
make run
```

## Glow Prompt

- 可直接复制的 glow 引入提示词：[`docs/glow_prompt.md`](./docs/glow_prompt.md)

## Core Features
- Provider-agnostic deployment spec
- EAS adapter (first provider)
- REST + MCP endpoints
- Image lifecycle management
- Ops aggregation for metrics/logs/cost
